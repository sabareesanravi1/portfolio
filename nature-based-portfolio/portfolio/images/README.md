# Images Folder

Place your images here. Suggested files:

- `avatar.jpg` — Your profile photo (used in the Personal Intro section)
  - Replace the "SR" initials in `index.html` with: <img src="images/avatar.jpg" alt="Sabareesan Ravi">

- `og-image.jpg` — Open Graph preview image (1200×630px recommended)

- `favicon.ico` — Browser tab icon

- `youtube-thumbnail.jpg` — Optional: YouTube channel banner/thumbnail

## Usage in HTML

Reference images like this:
```html
<img src="images/your-image.jpg" alt="Description">
```
